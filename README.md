# SIGAP Gizi - Sistem Informasi Gizi Anak dan Ibu Terpadu

## Tentang Proyek
**SIGAP Gizi** adalah aplikasi web modern berbasis Single Platform with Dual Access System yang dirancang untuk mengelola data gizi, memberikan rekomendasi menu bergizi, serta monitoring status nutrisi anak dan ibu (hamil & menyusui) secara real-time.

---

## Fitur Utama

### Customer (User) Portal
*   **Rekomendasi Menu:** Melihat rekomendasi menu makanan harian berdasarkan data antropometri dan skrining risiko pengguna.
*   **Jadwal Menu Harian MBG:** Menampilkan jadwal menu Makan Bergizi Gratis (MBG) lengkap dengan analisis gizi secara *real-time* serta kolom *feedback* dari orang tua.
*   **Edukasi Gizi:** Pengguna mendapatkan akses artikel dan informasi edukasi harian yang konsisten dan berbasis bukti.
*   **Konsultasi Gizi Online:** Pengguna dapat berkonsultasi langsung dengan dietisien/ahli gizi profesional secara mudah melalui sistem *chat*.


### Administrator Dashboard
*   **CRUD Menu Harian:** Mengelola seluruh data jadwal menu harian MBG berdasarkan lokasi dapur/sekolah.
*   **CRUD Edukasi Gizi:** Mengelola dan mempublikasikan seluruh data artikel edukasi kesehatan.
*   **Monitoring Data & Konsultasi:** Memantau semua aktivitas sistem, data pengguna, serta riwayat *chat* konsultasi yang masuk ke dalam *database*.

---

## 🛠️ Teknologi yang Digunakan
*   **Bahasa Pemrograman:** JavaScript (JSX)
*   **Frontend Framework:** React (Vite)
*   **Styling & Animasi:** Tailwind CSS
*   **Database & Auth:** Supabase (PostgreSQL)
